เพราะว่าผมเห็น browser ออกใหม่ เลยลองโหลดมาใช้ดูเฉย ๆ

## ความประทับใจแรก

Opera One เป็น browser ที่ค่อนข้างแปลกสำหรับ ด้วยความที่ว่าทางโน้นบอกว่าเป็นการ re-design ผมก็เลยเข้าใจได้อยู่
จะเห็นว่าแถบด้านซ้ายจะมี ChatGPT, ChatSonic (คล้าย ๆ ChatGPT แต่คนละเว็บ) มีกระทั่ง Messenger กับ WhatsApp ซึ่งก็น่าสนใจอยู่

## ความลื่น

จากการใช้ Opera One ภายในหนึ่งวัน ผมมองว่าเป็น browser ที่ลื่นพอสมควร สมกับการที่บอกว่าเกิดมาแก้ปัญหาเรื่อง performance จาก UI
rendering เลย

## ฟีเจอร์

ผมไม่ได้ใช้เป็น browser หลัก ๆ ผมก็เลยไม่ได้สนใจว่าจะมี features อะไรน่าสนใจไหม แต่ว่าก็มีการแนะนำพวก AI Prompt ถ้าเราลากเมาส์คลุมข้อความ
มันก็จะขึ้นคำสั่งออกมา ผมมองว่าเป็นการรบกวนมากกว่า ผมไม่ได้ใช้ AI prompt เยอะเลยปิดไป ก็ถือว่าเป็นเรื่องที่ดีที่ผมสามารถปิดมันได้

Opera One เป็น Chromium ทำให้ผมสามารถโหลด extension มาใช้งานได้ ถ้าเกิดว่าผมจะใช้เป็นตัวหลักจริง ๆ ก็ไม่ได้มีปัญหา แต่ตอนนี้ขอผ่าน
เพราะว่า Opera One ก็ยังเป็นแค่ public beta เท่านั้น

## โดยรวม

ถือว่าเป็นของใหม่ที่ค่อนข้างดีทีเดียว ผมกะว่าจะรอให้ stable แล้วมาดูอีกทีนึง
